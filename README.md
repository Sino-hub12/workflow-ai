# WorkFlow AI

An AI-powered Workplace Productivity Assistant designed to automate repetitive administrative tasks such as email drafting, meeting note summarization, task planning, workplace research, and AI chatbot support.

## Features
- Smart Email Generator
- Meeting Notes Organizer
- AI Task Planner
- Research Assistant
- Workplace AI Chatbot

## Technologies Used
- Lovable AI
- ChatGPT
- GitHub
